# Advarsarial Attack Tester

Implementation of adversarial attacks and defenses for deep learning models.

## Structure
- `attacks/`: FGSM, PGD, C&W attacks
- `defences/`: Adversarial training, TRADES, MART
- `evaluation/`: Robustness evaluation metrics
- `train/`: Training scripts

## Attacks Implemented
- FGSM (Fast Gradient Sign Method)
- PGD (Projected Gradient Descent)
- C&W (Carlini & Wagner)

## Defenses Implemented
- Standard Adversarial Training
- TRADES
- MART
